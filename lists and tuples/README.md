# 📂 Lists and Tuples (Python Practice)

This folder contains basic Python programs to practice **lists and tuples** concepts.  
These programs are part of my learning journey in Python.

## 📁 Files Included

### 1. ifElse.py
- Demonstrates the use of **if-else conditions**
- Helps understand decision-making in Python

### 2. listCount.py
- Counts elements in a list
- Practice with built-in functions like `count()`

### 3. listOperation.py
- Covers basic list operations such as:
  - Adding elements
  - Removing elements
  - Updating values
  - Traversing lists

### 4. tupleSort.py
- Works with tuples
- Demonstrates sorting (by converting tuple into list if needed)
